Dungeon Crawler

Cloner le repo sur votre machine
Ouvrir le projet dans VSCode et simplement exécuter le main.py pour que le jeu s'ouvre.

Si votre nom d'utilisateur existe, vous pourrez retourner dans votre session, sans même devoir cliquer sur une classe de personnage.

Sinon, pour créer un personnage, simplement lui donner un nom et choisir une classe.
